# Project1 -- Hotel Booking Management

酒店预订管理系统，基于 React + Vite + Supabase 构建。

## 技术栈

- React 19 + Vite 8
- React Router v6
- TanStack React Query v4
- Styled Components v6
- Supabase (认证 + 数据库)
- Recharts (图表)
- React Hook Form
- React Hot Toast

## 功能

- 用户认证（登录/注册/登出）
- Dashboard 数据仪表盘
- 预订管理（增删改查）
- 入住/退房操作
- 客舱管理
- 用户设置
- 深色模式

## 启动

```bash
npm install
npm run dev
```
